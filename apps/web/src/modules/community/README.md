# Community Module

Social features including posts, comments, and weekly topics.

## Features
- Community feed with threaded posts
- Post creation with images and tags
- Comments with likes
- Weekly topic discussions
- Post interactions (like, comment)

## Structure
- `application/` - Feed, comments, interactions hooks
- `infra/` - Community API client
- `model/` - Community types
- `styles/` - Community-specific styles
- `ui/` - Feed, composer, comments components
