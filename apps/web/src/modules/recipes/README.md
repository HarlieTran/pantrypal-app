# Recipes Module

## Responsibility
Fetch recipe suggestions/details and manage recipe cook preview/apply flows.

## Public API
- `RecipesPage`
- `fetchRecipeSuggestions`
- `fetchRecipeDetails`
- `cookRecipe`
- `RecipeSuggestion`
- `RecipeDetails`
- `CookRecipeResult`

## Dependencies
- Backend HTTP API via `infra/recipes.api.ts`
