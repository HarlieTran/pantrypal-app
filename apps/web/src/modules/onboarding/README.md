# Onboarding Module

User onboarding flow for dietary preferences and recipe selection.

## Features
- Dietary questionnaire (diet type, allergies, preferences)
- AI-powered recipe preference picker
- Profile completion workflow

## Structure
- `application/` - Questionnaire and picker hooks
- `infra/` - Onboarding API client
- `model/` - Onboarding types
- `styles/` - Onboarding-specific styles
- `ui/components/` - Questionnaire and picker components
